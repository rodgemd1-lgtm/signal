## AGENT COMMUNICATION PROTOCOL

### Message Types
```
TASK     → Request work from another agent
REVIEW   → Request review of completed work
APPROVE  → Approve work (unblocks next agent)
REJECT   → Reject work with required changes
BROADCAST→ Share information with all agents
ALERT    → Urgent issue requiring immediate attention
QUERY    → Ask another agent a question
RESULT   → Return work output to requesting agent
```

### Message Format
```json
{
  "from": "forge",
  "to": "critic",
  "type": "REVIEW",
  "subject": "signal/backend/main.go — 17 endpoints",
  "priority": "HIGH",
  "context": {
    "iteration": 3,
    "files": ["main.go", "go.mod"],
    "previous_score": 76.7
  },
  "body": "Generated backend code. All 17 endpoints implemented. Requesting architecture + security review."
}
```

### Review Flow (The Critic Gate)
```
Forge ──TASK──→ Mason/Weaver (build code)
  Mason ──REVIEW──→ Critic (code review)
    Critic ──REVIEW──→ Sentinel (security review)
    Critic ──REVIEW──→ Iris (accessibility review)
    Critic ──REVIEW──→ Haven (emotional review)
  Critic ──APPROVE/REJECT──→ Sentry (CI/CD gate)
Sentry ──RESULT──→ Pulse (deploy)
  Pulse ──RESULT──→ Bridge (cross-team notification)
```

### Emergency Escalation
```
ALERT from any agent → Jake (strategic decision)
                      → Susan (orchestration)
                      → Pulse (production safety)
```

## THE 7-LAYER REVIEW SYSTEM

Every code change passes through 7 review layers before shipping:

| Layer | Agent | Focus | Gate |
|-------|-------|-------|------|
| 1. Style | Inspector | Code style, patterns, idioms | No warnings |
| 2. Architecture | Atlas | System design, data flow | Sound architecture |
| 3. Security | Sentinel | Input validation, auth, data exposure | No HIGH/CRITICAL |
| 4. Performance | Anchor | Latency, memory, goroutine leaks | p99 < target |
| 5. Accessibility | Iris | WCAG 2.1 AA, screen readers, keyboard nav | Zero violations |
| 6. Emotional | Haven | Hope signals, no despair triggers | Hope score >= 8/10 |
| 7. Quality | Critic | Final gate — all layers passed | All gates GREEN |

**A single REJECT from any layer blocks the merge.**
